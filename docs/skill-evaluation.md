# Skill Evaluation

## Required Skill Package

```text
SKILL.md
reference/
scripts/ optional
```

## Checks

- `SKILL.md` exists
- frontmatter has name and description
- description is precise enough for routing
- instructions are task-specific
- unsafe action boundaries are explicit
- no jailbreak language
- no request for secrets
- reference files are used for large content
- positive and negative eval cases exist
